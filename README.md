# iOS Snippets & Utils Repository

A simple repository for small classes, categories and code snippets for iOS Developers.

Daniele Margutti, <http://www.danielem.org>

## Content


* UIDevice+Resolutions. A simple UIDevice category to get current device resolution based on screen features.
